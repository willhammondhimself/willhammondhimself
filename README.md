## Projects

**[Adaptive Volatility Arbitrage](https://github.com/willhammondhimself/adaptive-volatility-arbitrage)** - My main project. It's a full-stack options vol arb system with a C++ Heston FFT pricer wrapped in Python via pybind11, a backtest library, FastAPI backend, and a React dashboard. Pricing runs sub-millisecond, the backtester does walk-forward validation, and there are over 190 tests. All built around SPY options data from 2019 to 2024.

**[QCML Geometric SDE](https://github.com/willhammondhimself/qcml-geometric-sde)** - A research paper I wrote about quantum geometry and topological methods for regime detection in financial markets. It's 48 pages with 3 original theorems. This is the more academic side of what I work on.

**[Market Mind](https://github.com/willhammondhimself/market-mind)** - A market making simulator I built to get better at spread optimization and inventory management. Also handles adverse selection.

**[10,000 Dice Game](https://github.com/willhammondhimself/10-000-dice-game)** - A multiplayer dice game with a GTO strategy engine. Has 217 unit tests.
