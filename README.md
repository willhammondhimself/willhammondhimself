## Research

Selected poster at the 2026 APS Global Physics Summit. Manuscript in preparation.

## Projects

**[MarketMind](https://github.com/willhammondhimself/market-mind)** - Multi-agent autonomous system with 7 AI agent types (rule-based, reinforcement learning via SAC, evolutionary, and LLM-powered). Agents independently observe, plan, coordinate, and adapt. Built with FastAPI, React/TypeScript, WebSocket streaming, and a price-time priority matching engine.

**[Market Population Dynamics](https://github.com/willhammondhimself/market-population-dynamics)** - Agent-based simulation engine written in Rust with PyO3 Python bindings. Three behaviorally-distinct agent populations with replicator dynamics. Maps the full population simplex to find phase boundaries and bifurcation conditions.

**[Adaptive Volatility Arbitrage](https://github.com/willhammondhimself/adaptive-volatility-arbitrage)** - Full-stack options vol arb system. C++ Heston FFT pricer (pybind11), Gaussian Mixture Model regime detection, walk-forward backtester, FastAPI backend, React dashboard. 190+ tests.
